# 🟣 SendGo - Moroccan E-Commerce Delivery Platform

SendGo is a modern, multilingual platform that streamlines last-mile delivery services for e-commerce sellers across Morocco.

## 🌍 Features

- 🌐 Multilingual interface (Arabic 🇲🇦 | French 🇫🇷 | English 🇬🇧)
- 📦 Real-time order tracking with tracking number
- 🚚 Live courier tracking with Google Maps
- 📊 Dashboard with delivery performance analytics
- 🧑‍💼 Merchant and client portals
- 🎯 Ranking and challenges system for couriers
- 📡 Admin panel with full control
- 📬 SMS/email notifications

## 🎯 Tech Stack

- Frontend: React.js + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB (or Firebase)
- APIs: Google Maps API, Twilio (SMS)

## 📂 Project Structure

/client       # Frontend React app  
/server       # Backend API  
/public       # Static files

## 🚀 Installation

git clone https://github.com/HAMZAKARROUMI/sendgo-delivery-platform.git  
cd sendgo-delivery-platform

cd client && npm install && npm start  
cd ../server && npm install && npm run dev
